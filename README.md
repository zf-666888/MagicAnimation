##教程开始：

###第一步：PCmask模拟机型
 -重启电脑进入BIOS模式
`这里不对如何进入BIOS模式作过多解释，不会的百度或者问一下客服你的电脑如何进入BIOS模式`
 - 关闭安全引导启动
  - 进入BIOS后找到Security下的Security Boot，把Enable改成Disable，F10保存退出正常启动电脑。
 - 打开第二步文件夹中心情助手维护文件夹，右击XQZS3.1(怀念版).exe以管理员身份运行，听到余承东的Super Fast Charging的语音播报即打开成功，不必惊慌。
 - 找到安装专区中的PC Mark，确保安全引导已关闭的情况下，选择初次模拟成荣耀笔记本，然后开始模拟。（心情助手来源于B站up主又菜又爱玩的心某人）
 - 等待模拟环境，模拟成功之后后提醒你重启电脑，重启电脑即可，重启电脑后，你的电脑部分信息会更改为荣耀笔记本HGE-WX6 （百度得知这是荣耀 MagicBook V 14）。
  - 这时候你的电脑已经变成了荣耀笔记本，可以对荣耀电脑管家安装包随便安装，
 - 打开荣耀电脑管家，检测更新，把第一步下载的最新荣耀电脑管家安装包20.0.0.35自动覆盖安装即可。

###第二步：安装过渡的官方最新管家
 - 打开第三步文件夹中PCManager_2IN1 20.0.0.45(SP12C233)-Software-PCManager_Setup_2IN1_20.0.0.45(SP12C233)，直接双击运行即可，这是荣耀官网最新的荣耀电脑管家的安装包，再次进行覆盖安装，安装完成之后打开进入首页，手动下载YoYo助理，然后随便玩一玩就有控制中心了，到这里所有的荣耀组件已经全部有了，接下来就是Magic视界。

###第三步：安装内测最新管家
 - 打开第四步文件夹中PCManager_Setup_4IN1_20.0.0.48(SP4C233)，直接双击打开运行即可，此文件来源于酷友XX提取，非常感谢。
 - 覆盖完这个版本的荣耀电脑管家就结束了，此时，你就拥有了一个完全体荣耀电脑管家和Magic视界。
 - 接下来你就可以打开Magic视界进行畅通的使用啦！

##目前我遇到的问题：
 荣耀工作台、YoYo助理、控制中心中的大部分功能都可以使用
 荣耀笔记和荣耀文档也能同步你荣耀云上的文档显示，如果遇到荣耀笔记闪退情况，卸载重装即可解决。
 控制中心中的截屏选词和智慧搜索是用不了的，他会告诉你需要更新，但是又找不到更新包，可能因为这两个和YOYO助理绑定了，20.0.0.45版本提供的YOYO助理不支持这两个功能，只要找到内测包给他覆盖安装应该能解决这个问题。
 再聊聊Magic视界，Magic视界的UI是和微软本身桌面分开的UI，从酷友XX方法中可以知道，运行文件就可以启动，在Win10的时候都能直接套上这个UI，但服务不完整。
 经过这个教程得到的荣耀Magic视界服务基本上全部有了，目前还没有碰到什么bug，属于是正规军了。
 在设置里面可以调节动效和图标，桌面右击可以加卡片和收纳图标（桌面整理）但是有点鸡肋，全部收集下面右下角那个框框里了，而且框框不可调节
