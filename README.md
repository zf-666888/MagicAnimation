## 教程开始

### 第一步：PCMask 模拟机型

- 重启电脑进入 BIOS 模式
  - 这里不对“如何进入 BIOS 模式”作过多解释；不会的请百度，或咨询客服你的电脑如何进入 BIOS。
- 关闭安全引导（Secure Boot）
  - 进入 BIOS 后找到 `Security` 下的 `Security Boot`，把 `Enable` 改成 `Disable`，按 `F10` 保存退出并正常启动电脑。
- 打开第二步文件夹中的“心情助手维护文件夹”，右击 `XQZS3.1(怀念版).exe` 以管理员身份运行
  - 听到余承东“Super Fast Charging”的语音播报即打开成功，不必惊慌。
- 找到安装专区中的 `PC Mark`
  - 确保安全引导已关闭的情况下，选择“初次模拟成荣耀笔记本”，然后开始模拟。
  - （心情助手来源于 B 站 up 主 @又菜又爱玩的心某人）
- 等待模拟环境
  - 模拟成功之后会提示你重启电脑，重启即可。
  - 重启后，你的电脑部分信息会更改为荣耀笔记本 `HGE-WX6`（据百度得知这是荣耀 MagicBook 的某机型）。
- 此时即可安装荣耀电脑管家
  - 打开荣耀电脑管家 → 检测更新 → 把第一步下载的最新荣耀电脑管家安装包 `20.0.0.35` 自动覆盖安装即可。

### 第二步：安装过渡的官方最新管家

- 打开第三步文件夹中：
  - `PCManager_2IN1 20.0.0.45(SP12C233)-Software-PCManager_Setup_2IN1_20.0.0.45(SP12C233)`
- 直接双击运行即可（这是荣耀官网最新的荣耀电脑管家安装包）。

### 第三步：安装内测最新管家

- 打开第四步文件夹中：
  - `PCManager_Setup_4IN1_20.0.0.48(SP4C233)`
- 直接双击运行即可（此文件来源于酷友 XX 提取，非常感谢）。
- 覆盖完此版本后就结束了：
  - 此时你就拥有了完全体荣耀电脑管家和 Magic 视界。
  - 接下来就可以打开 Magic 视界进行使用啦！

## 目前我遇到的问题

- 荣耀工作台、YOYO 助理、控制中心中的大部分功能都可以使用。
- 荣耀笔记和荣耀文档也能同步荣耀云上的文档显示。
  - 如果遇到荣耀笔记闪退，卸载重装即可解决。
- 控制中心中的“截屏选词”和“智慧搜索”不可用：
  - 会提示需要更新，但找不到更新包。
  - 可能因为这两个功能与 YOYO 助理绑定；`20.0.0.45` 版本提供的 YOYO 组件可能不匹配。
- 关于 Magic 视界：
  - UI 是和 Windows 桌面分开的（独立 UI）。
  - 从酷友 XX 的方法可知，运行文件即可启动；Win10 时可直接套上 UI，但服务相关仍需完整安装。
  - 目前服务基本齐全，还没遇到明显 bug，属于“正规军”。
  - 设置里可调节动效和图标；桌面右击可加卡片和收纳图标（桌面整理），但比较���肋：
    - 会把内容收集到右下角的框框里
    - 框框不可调节
